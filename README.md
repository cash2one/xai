###pyliterature
xai is a AI program. xai can search and read article, get data and write new code by itself.



###Author

###Dependencies

* Python >=27
* nltk



####Examples

```python
>>> from xai import AI
>>> XAI = AI('Xing', 'male')
>>> XAI.start()
>>> XAI.life.learn.word(filename = 'word/cambtionary.dat')
```
